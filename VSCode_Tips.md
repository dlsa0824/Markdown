# VSCode Shortcut Tips

---

### Open Command Pallete

> Ctrl + Shift + P

### Open New Window

> Ctrl + Shift + N

### Zoom VSCode size

> Ctrl + '+' \
> Ctrl + '-'

### Open File

> Ctrl + O

### Quick File Open

> Ctrl + P

### Change File Tabs

> Ctrl + PageUP \
> Ctrl + PageDown

### Spilt Editor

> Ctrl + \

### Go To Start & End Of File

> Ctrl + Home
> Ctrl + End

### Remove Line

> Ctrl + Shift + K

### Change Language Mode

> Ctrl + K -> M

### Open Terminal

> Ctrl + `

### Open Newe Terminal

> Ctrl + Shift + `

### Change VSCode Theme

> Ctrl + K -> Ctrl + T
